Instructions

Learn how to play in minutes.

How to play

  1. Navigate to the Game page using the menu above.
  2. Enter your name in the text field on the game page.
  3. Press Start — the AI will flash a sequence of shapes.
  4. Watch carefully and memorize the order in which the shapes light up.
  5. Once the flashing stops, click the shapes in the same order to match the sequence.
  6. Each correct round adds one more shape to the sequence — it keeps getting harder!
  7. The game ends when you click the wrong shape. Try to beat your score!
  8. Use the Theme buttons in the nav to change the page color scheme.
  9. Visit About Us to learn more about the team behind this game.
