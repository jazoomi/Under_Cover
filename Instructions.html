How to play

This is the instructions page for Matching game!
to first access to the game, naviagte to "Game" with the navigation at the top.
once you become navigated to the page, this where is you can start the game.
The game consists of 4 shapes where you would need to remember the sequence that we randomly generate for you.
you can get enter your name to be displayed on the for when the game is finished and you can press start to start the game.
The game will start and flash some shapes for you to remember. When shapes stop flashing, thats when you make an attempt in following the pattern given to you.
The game has been coded with a highly intelligent AI so that it randomly chooses shapes in a sequence.
The game ends only when you get an incorrect shape. Furthermore, you can press start to start a new game after.

as additonal features, you can press the buttons at the top to change the theme of the page. For whichever best suits you.
