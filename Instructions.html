Instructions

How to play:

This is the instructions page for Matching game!
to first access to the game, naviagte to "Game" with the navigation at the top.
once you are navigated to the page, this where is you can start the game.
the game consists of 4 shapes where you would need to remember the sequence that we randomly generate for you.
you can get enter your name to be displayed once the game is finished
press "start" to start the game.
the game will start by flashing some shapes you must remember.
once the shapes stop flashing, thats when you make an attempt in following the pattern given to you.
this game has been coded with a highly intelligent AI so that it randomly chooses shapes in a sequence.
The game ends only when you get an incorrect shape.
furthermore, you can press start to play again.
as additonal features, you can press the buttons at the top to change the theme of the page (a selection of blue, green, and yellow).
